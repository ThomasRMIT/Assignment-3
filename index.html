Project Description

Topic

We want to create a simple-to-operate program for users that can increase security on their systems to defend a system from intrusion, stop personal data being leaked through websites, emails, mobile data storage and much more! STEVE, being a simple-to-operate program, is also a tool for learning, STEVE will be equipped with a library of known scams and tactics, as well as the ability to recognise them. STEVE is able to alert the user to what the detected intrusion attempt is doing, and provide instructions on how to stop it or have an option to enable STEVE to defend against such intrusions automatically, whilst providing feedback and the process taken.
We want STEVE to educate users against the tactics that hostile parties would use against them. STEVE would protect the user, but ultimately; any security system is only as strong as its weakest link. Sometimes, the problem exists between the keyboard and the chair. But with STEVE, hopefully society will become more wary of scammers and phishers.

Motivation

A world free of corruption, of deceit, and of fear is a beautiful vision. Stop Technologically Educated Villains Embezzling (STEVE) could help achieve that. Some might say it’s a bit ambitious, but it’s within reach. Security may not be that ‘sexy’, when it comes to IT trends; but it’s a necessity. Protecting the user from scams, virus and phishing is important work; and if we’re able to do it, it’ll show future employers that we can stick with an essential, long-term project. It’ll also show off our skills as a team, and individuals.

Landscape

Many competitors exist for *one* facet of STEVE. The anti-virus aspect of it is something that’s been done many times, and done pretty well. Sure, systems are still infected; but every system is only as strong as its weakest point. That’s where our difference comes into play. STEVE is a tool for *educating* the user; so they’re less likely to click on suspicious links or download untrustworthy attachments. It might teach them to improve their passwords, and be more conscious about to whom they’re giving their information. It might even teach them better security practices, locking their computer/device if they have to leave it unattended for a short while.

Skills and Jobs

Introduction

Upon receiving $1,000,000 funding for our project our small little group decided it was time to expand. As we needed extremely qualified professionals we have decided that we will be looking to fill four crucial roles. These roles are: Project Manager, Software Engineer, Penetration tester and UI designer.

Project Manager

The project manager for STEVE will oversee all aspects of this software project. We will need someone who can set deadlines, assign responsibilities and monitor and summarize progress of project. The project manager will need to have great leadership and communication skills in order to ensure that the team is working as efficiently as possible. The project manager will also report back to the executives in our business (which will be us) and give us a report on how the development is going and if changes need to be made. Our project manager will need to have an understanding of multiple coding languages and will need to have experience managing previous projects in the past. This role is probably the most crucial role as it requires a lot more than a skill set but rather a very strong leader that can delegate tasks as necessary with effective communication.

Software Engineer

A software engineer will be the backbone to our software’s development. They will do all the hard nitty gritty word of actually coding the software and making sure that it is working as it should. The Software engineers that we are looking for need to have Java basic technology, WEB front-end technology, database technology, big data technology, and .NET technology. We will need around 2-3 software engineers for this development as there is a large amount of code that needs to be written and perfected.

Penetration tester

The penetration tester will have indeed a very important role. We will need a highly skilled security specialist that will spend many hours attempting to breach our computer and network security system. As we are a security company it is of the most utmost importance that our product will be able to deliver an extremely high-end security solution without any flaws. The penetration tester will work from different locations and networks, and on different devices in order to understand how impenetrable or penetrable our system is.

UI Designer

The UI designer will have a role of creating amazing user experiences. Ideally, they should have an eye for clean and artful design. We will need someone experienced in the design and computing field, as they will be designing the direct to market interface. UI is very important in our application as no matter how powerful the security is, it is a vital part of the application that our security system will also teach the user of known risk and how to prevent them. If this learning process does not have a clean crisp interface the user will not be interested in learning thus having a qualified experienced UI designer is vital.
